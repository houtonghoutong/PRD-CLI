## 🔐 如果提示需要 OTP（6位验证码）

### 情况 1：你有手机认证器应用

1. **打开手机上的认证器应用**
   - Google Authenticator
   - Microsoft Authenticator
   - Authy
   - 等等

2. **找到 npm 的条目**，会显示6位数字

3. **重新运行并带上验证码：**
   ```bash
   npm publish --otp=123456
   ```
   （把 123456 替换成你看到的6位数字）

---

### 情况 2：你没有认证器应用

**使用这个命令生成访问令牌：**

```bash
# 在浏览器中打开（如果网站恢复正常）
open https://www.npmjs.com/settings/houtong/tokens
```

或者暂时跳过 2FA：

1. 访问 https://www.npmjs.com/login
2. 登录后访问 https://www.npmjs.com/settings/houtong/tfa  
3. 临时禁用发布时的 2FA 要求
4. 然后直接运行 `npm publish`
5. 发布后再启用回来
