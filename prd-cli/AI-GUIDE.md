## 📂 规则文件位置

| IDE | 规则文件 | 说明 |
|-----|---------|------|
| **Cursor** | `.cursorrules` | 完整规则 |
| **Antigravity** | `.antigravity/rules.md` | 完整规则 |
| **其他 AI** | `AI-PROMPT.md` | 通用提示词 |

## 🚨 开始工作前的自我检查

```
□ 我是否已阅读规则文件？
□ 我是否知道当前项目状态？（查看 .prd-config.json）
□ 我是否会通过对话引导 PM，而不是直接填写？
□ 我是否会让 PM 做决策，而不是替 PM 决定？
```

**如果任何一项是"否"，停止并先阅读规则！**
